# BuildingWordBattle
建築伝言ゲーム

## 仕様リスト
- [ ] Paper1.12.2 + OpenJDK17
- [ ] 途中参加は自動で観戦(スペク)へ
- [ ] 出題者は180秒 (configで変更可能)
- [ ] 出題は1人1回、回答者の中からランダム
- [ ] ポイントは(出題時間 - 経過時間)
- [ ] 放送主はコマンドでスタート前に設定
- [ ] 放送主はDiscordの専用チャンネルに出題
- [ ] 出題者は右側のスコア、残り時間は上のボスバー(まあこれは任せる)
- [ ] 個人戦とチーム戦
